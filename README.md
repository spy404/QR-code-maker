<h1 align="center">
  QR code maker
</h1>

Steps to install:
- <pre> git clone https://github.com/spy404/QR-code-maker.git </pre>
- <pre> cd QR-code-maker </pre>
- <pre> pip install -r requirements.txt </pre>
- <pre> python3 ./main.py </pre>
